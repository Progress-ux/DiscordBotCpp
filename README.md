# DiscordBotCpp

### Копирование моего репо
```shell
git clone https://github.com/Progress-ux/DiscordBotCpp.git
cd DiscordBotCpp
```

### Добавление DPP
```shell
mkdir libs
cd libs
git clone https://github.com/brainboxdotcc/DPP.git
cd ..
```

### Создание папки build 
```shell
mkdir build 
cd build 
```
### Сборка бота
```shell
cmake ..
cmake --build .
./main
```

### Возможные команды 
```shell
sudo pacman -S cmake
```
